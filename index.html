"T.O.P. Sign-Up Form"


Rather than make a boring sign-up form, I created an imaginary lecture sign-up form with way too many fun "Lovecraftian" horror story references. (Mobile ready.)

"T.O.P. Calculator"


This project hews closer to the basic project parameters of a calculator. However, I had to add extra fun buttons, like a Roman numeral converter, and a "The Meaning of Life" button. (Mobile ready.)

"T.O.P. Etch-a-Sketch"


This project started off following the project brief of creating an etch-a-sketch, but it quickly morphed in a bigger project as I became obsessed with making a TTRPG dungeon-maker that also happened to be an etch-a-sketch. (Not mobile ready.)

"T.O.P. Rock-Paper-Scissors"


This was one of my favorite T.O.P. projects. Rather than make a "rock-paper-scissors" type game like everyone else, players in my game choose one of the mystical elements of Wood, Fire, Water, Earth, and Metal. (Not mobile ready.)

"T.O.P. Landing Page"

"T.O.P. Recipe Page"


An early Odin Project HTML challenge. It's pretty basic stuff. Set an alarm in case you fall asleep. (Not mobile ready.)

"F.C.C. Pomodoro Clock"


This was a fun coding challenge that I wrote last year for Free Code Camp (F.C.C.). It's also useful for time management. This is definitely code that I need to revisit and tinker around with, as it functions a wee bit differently depending on the browser. Once again, the Linux Chrome browser won't display this project's emoji images or end alarm sound properly. (Mobile ready.)

One of my favorite older apps that has some humorous "local pub conversation utility." Sadly, this app definitely breaks if using a Linux Chrome browser. (Mobile ready.)
